Pick a layout

Three real layouts, same content in each. They are structurally different, not just recolored. Scroll each panel. Pick the one you want, then follow Step 2 in docs/lesson-4b-tailwind.md.

All three use identical section names, so switching later is the same copy-and-paste. You are not locked in.

Fun

Rounded, asymmetric corners, icons, colorful cards. Friendly and informal.

File: template/layouts/fun.html

Serious

Tight, traditional, conservative. Reads like a professional CV.

File: template/layouts/serious.html

Bold

Dark, editorial, oversized type. Looks like a magazine spread.

File: template/layouts/bold.html

Not sure?

  • Serious if you want it to look like a resume you would hand to an employer.
  • Fun if you want it to feel like you rather than a document.
  • Bold if you want people to react to it the second it loads.

There is no wrong answer, and switching later costs about five minutes.
