# Summer Web Basics

Welcome! Over the next few weeks you'll learn just enough web development to
build your own personal resume site — real HTML, CSS, and JavaScript, using
the same tools professional developers use every day.

## How this works

- Everything you need lives in this repo, one lesson at a time.
- When a lesson is ready, run `git pull` to get it.
- Each lesson adds a new file in `docs/` — read it, then do the task inside it.
- Don't peek ahead — the lessons build on each other on purpose.
- After every lesson, jot a couple sentences in `reflections.md` — what
  clicked, what didn't. Takes a minute, and it's yours to look back on.
- At a few key lessons, you'll save a screenshot to `progress-screenshots/`
  — by the end you'll have your own before/after story of the whole month.

## What you'll end up with

A personal website with:
1. A resume page (your projects, skills, background)
2. A second page that's entirely your own idea
3. A shared homepage the whole group builds together at the end
4. (Optional) A real live URL on the internet, hosted the same way real
   companies host their sites

## Quick privacy reminder

Everything you write here can end up public. Stick to what you'd normally
put on a resume — no home address, phone number, or anyone else's personal
info. When in doubt, leave it out.

## Do not push here

This repo is for lessons only. If you try to push and get an error, that's
expected — see Lesson 1 for where your own work actually goes.
